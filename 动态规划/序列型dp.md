## [数组跳跃](https://leetcode-cn.com/problems/jump-game/)
题目大意：给定一个数组，每个数组的值代表在当前点向后跳的最大长度，问能否跳到最后一个数。  
分析：这是一个序列存在dp。  
定状态：  
最后一步：如果能跳到第n个数，我们考虑他的最后一步，一定是从第i个位置跳过来的，故此时有num[i] + i >= n  
原问题：是否能跳到n     
子问题：是否能跳到i  
状态dp[i]代表是否跳到i  
求方程：  
dp[i] = dp[j]&&(num[j]+j >= i) 其中j<i  
判边界：  
dp[0] = true
